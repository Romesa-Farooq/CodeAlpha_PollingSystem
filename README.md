# Polling System Smart Contract

## Description
This project is a blockchain-based polling system developed using Solidity. It allows users to vote for predefined candidates while securely storing vote counts on the Ethereum blockchain. The smart contract demonstrates the use of arrays, mappings, constructors, and functions in Solidity.

## Features
- Create a list of candidates
- Cast votes for a candidate
- Store vote counts securely using mappings
- Retrieve the total number of candidates
- Read candidate names and vote counts
- Deploy and test on Remix IDE

## Technologies Used
- Solidity
- Remix IDE
- Ethereum Virtual Machine (EVM)

## Platform
- Ethereum Blockchain (Remix VM)

## How to Run
1. Open Remix IDE.
2. Create a new file named `PollingSystem.sol`.
3. Paste the Solidity code into the file.
4. Compile the contract using Solidity version 0.8.x.
5. Deploy the contract using Remix VM (Cancun).
6. Test the voting functions and verify the results.
